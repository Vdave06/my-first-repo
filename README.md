## How to run

This project requires Python 3.6+. To run the included example script:

```bash
# from the repository root
python3 scripts/hello.py
```

Optional: create and activate a virtual environment before running:

```bash
python3 -m venv .venv
source .venv/bin/activate   # macOS / Linux (zsh)
pip install -r requirements.txt  # if requirements.txt exists
```
